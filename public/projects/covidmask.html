Computer Vision

Covid Mask Detection

A high-accuracy real-time face mask detection system engineered to enhance public safety during the pandemic.

Stack

  • Python / OpenCV
  • TensorFlow / Keras
  • NumPy / SciPy

The Challenge

Developing a robust model that can correctly identify faces even when partially obscured and accurately classify the presence of protective gear in various lighting conditions.

Key Features

  • Real-time image attribute analysis
  • Partial face occlusion handling
  • High-precision feature extraction
  • Optimized inference speed
